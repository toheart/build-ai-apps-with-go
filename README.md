## 第 4 章骨架示例

`examples/04-go-react-fullstack/fullstack-starter/` 用来承接第 4 章
`AI 全栈技术栈上手（Go + React）`。

这一版示例的重点不是 AI 能力本身，而是先把下面几件事固定下来：

- Go 后端分层目录
- React 前端请求组织方式
- 统一接口返回结构
- YAML + 环境变量的配置方式
- 可持续复用的工程规范入口

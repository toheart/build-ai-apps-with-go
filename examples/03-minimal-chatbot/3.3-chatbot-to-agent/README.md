# 3.3 对应代码说明

这个目录存放 `3.3 从 Chatbot 到 Agent` 的示例代码。

## 文件说明

- `main.go`：程序入口、工具声明、最小 Agent 循环和命令行交互
- `go.mod`：Go 模块定义
- `.env.example`：示例环境变量

## 运行方式

1. 复制配置文件

```bash
cp .env.example .env
```

2. 修改 `.env` 中的模型服务配置

3. 运行程序

```bash
go run .
```

## 本阶段新增能力

相比 `3.2`，这一版代码新增：

- 工具描述注册
- 工具调用分发
- 工具结果回填后再次请求模型
- 最小 Agent 闭环日志输出

## 本阶段边界

这一版依然是“最小 Agent”，不建议提前加入：

- 复杂多工具编排
- 长期记忆
- 知识库检索
- 前后端分离界面
- 完整监控和评估体系

本阶段的目标只有一个：让读者真正理解“Tool Calling 是怎么把 Chatbot 变成最小 Agent 的”。
